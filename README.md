# Personal CV Website

## Project Information
This project is a personal CV website developed as part of a web development assignment.  
The website presents academic background, skills, projects, certifications, and contact information in a clean and professional layout.

The project is built using pure HTML and CSS, following Flexbox principles, and is fully responsive on both desktop and mobile devices.


## Technologies Used
- HTML5  
- CSS3  
- Flexbox  
- Git & GitHub  
- Vercel (Deployment)
